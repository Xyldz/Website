C#

Anders Hejlsberg the founder of JavaScript

C# is a C programming language derivative and is similar to C++. It uses the same basic operators as C++, is case sensitive, object oriented, and has almost identical syntax. There are some distinctions between C# and C++, however. Here are some examples:

The bool (boolean) data type is not recognized as an integer as it is in C++. But its used as a True or False data type, Global methods and variables are not supported in C#, while they are in C++, C# switch statements do not support fall-through from one case to another.

Most notably, C# is explicitly developed for the .NET Platform for Microsoft. This enables developers to take advantage of all the functionality the .NET API provides. It also means, however, that C# applications can run only on .NET runtime compatible platforms, such as Windows, Windows Server, and Windows Phone. The code needs to be compiled using a conversion tool like Microsoft .NET Native in order for programs written in C# to run on other platforms.
